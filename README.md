# joy-townsend-grc-portfolio
cybersecurity portfolio


## Overview
This repository contains employer-verifiable Governance, Risk, and Compliance (GRC) and cybersecurity portfolio projects.

## Purpose
The purpose of this portfolio is to document coursework, labs, and practical projects in a structured format that can be reviewed by instructors, recruiters, and employers.

## Portfolio Standard
Each submission includes:
- A README explaining the problem, what I did, evidence, and learnings
- Sanitized screenshots
- Artifacts or outputs
- Version history with dated changes
- Clear folder structure

## Projects
- Week 1 / Day 1 - Technology Foundations

## Review Method
Each project folder contains its own README, screenshots, artifacts, and a version history log so the work can be reviewed and verified quickly.
